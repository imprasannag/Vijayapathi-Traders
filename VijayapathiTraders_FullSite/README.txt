Vijayapathi Traders - Website package
How to use:
1. Unzip this folder.
2. Open index.html in your browser to preview the site locally.
3. Add your real images into assets/img/<category>/ folders, naming them as <category>1.jpg ... <category>12.jpg.
4. Replace banner.jpg in assets/img/banners/ with your banner image.
5. To publish online, upload all files to Netlify, Vercel or GitHub Pages.
